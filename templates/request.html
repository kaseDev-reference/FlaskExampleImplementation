HTTP Request Processing

This page shows how to process GET and POST requests on the backend with Flask.

GET Requests

The GET Method is a way of transmitting data to the server unsecurely through the url. It should never be used for data that needs to remain private like login credentials but it is ideal for data that influences the page in a meaningful way as the data is stored in the url and can be included in the link if shared.

To get an example of how a GET Request works, go to the url bar and add "?get_val=" to the end of the current url followed by whatever string you want. It will be displayed under this paragraph.

POST Requests

The POST Method is a way of transmitting data to the server in a secure fashion. It should be used in cases where the data in sensitive for if the data should not appear in the url for any reason. The most common use case for POST requests is form processing.
